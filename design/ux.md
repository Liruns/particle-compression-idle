## 1. 정보 구조 (IA) — 메인 탭 구성

```
[상단 탭바]
  ├── 압축      (메인, 기본 화면)
  ├── 연구      (D 사용 연구 트리)
  ├── 도감      (입자 수집 & 완성도)
  ├── 상전이    (조건 충족 시 점등 ●)
  └── 설정      (저장·오프라인·표기법 옵션)
```

### 탭별 진입 조건 및 우선순위

| 탭 | 해금 조건 | 핵심 정보 | 보조 정보 |
|---|---|---|---|
| 압축 | 시작부터 | r 반경 / 8단 체인 / 현재 층 메커니즘 | 자원 readout / 상태 로그 |
| 연구 | 원자층(L2) 진입 + 첫 D 획득 | A 가지 체인 증폭 노드 | B/C/D 가지 (층·조건 게이팅) |
| 도감 | 첫 입자 발견(게임 시작 즉시) | 층별 수집 현황 | 입자 상세·보너스 |
| 상전이 | **미지 서브층 벽(dec19~26) 도달 시에만 탭 ● 점등.** 알려진 물리 5층(dec0~19) 진입은 점등 없음 — 층 카드(스케일 진입·도감 항목·내러티브 비트)만 갱신 | QF 예정 / 리셋 매트릭스 / 타이밍 힌트 | 내러티브 비트 |
| 설정 | 시작부터 | 세이브·로드 / 오프라인 시각화 | 수치 표기법 전환(접두사↔과학) |

### 화면별 정보 우선순위 원칙

- **지금 중요한 것 하나**를 크게. 나머지는 수축된 섹션 또는 툴팁.
- 숫자는 항상 **과학 표기법(기본) / 접두사(옵션)**으로. 원시 float 절대 표시 금지.
- 방치형 핵심: 탭을 열지 않아도 상태를 아는 것이 목표. 탭 레이블에 간단 상태 표시.

## 2. 메인 화면 레이아웃 — 압축 탭

```
┌─────────────────────────────────────────────────────────────┐
│  [탭바: 압축 / 연구 / 도감 / 상전이● / 설정]  현재층·decade  │
├─────────────────┬───────────────────────┬───────────────────┤
│  LEFT PANEL     │   CENTER              │  RIGHT PANEL      │
│  (180px)        │   (가변)              │  (200px)          │
│                 │                       │                   │
│  자원 Readout   │  [ 반경 r 게이지 ]    │  현재 층 카드     │
│  E ─────────── │  원형·중심 글로우     │  층명·decade 범위 │
│  C ─────────── │  r 수치 (과학 표기)   │  짧은 설명 1줄    │
│  D ─────────── │  decade 진행 바       │                   │
│  QF ──────────  │                       │  [메커니즘 위젯]  │

│  생산 배율      │                       │                   │
│  QF 부스트      │  ── 8단 체인 ──       │  도감 진행도      │
│  공명 배율      │  T1 개수/비용/구매    │  (층 내 n/전체)   │
│                 │  T2 …                 │                   │
│  상태 로그      │  T3–T8               │  연구 가지 현황   │
│  (4줄 최신)     │  (미해금 시 흐림)     │                   │
│                 │                       │  상전이 힌트      │
│                 │                       │  (조건 충족 시)   │
└─────────────────┴───────────────────────┴───────────────────┘
```

### 2-A. 반경 r 게이지 — "작아짐=강해짐" 핵심 비주얼

**원형 게이지 설계:**
- 바깥 링: 현재 층 내 decade 진행도 (호弧 길이)
- 안쪽 동심원: 4개 고정 원 (희미, 구조용) — 작아질수록 중심에 가까운 원만 남는다
- 중심점: 강하게 빛나는 점(내부 발광색 #3ecf8e) — r이 작아질수록 글로우가 커짐
- 역설 구현: **r 수치(위)는 작아지고, 중심 글로우(아래)는 커진다**

**수치 표시:**
- r 값: `3.81×10⁻¹¹ m` (과학 표기, 항상)
- decade 진행 바: 하단에 분리 바 (`dec 1 ──[■■■□□]── dec 2`)
- 층 이름: 탭바 우측 상단에 고정 (`원자층 — dec 1.42`)

**마케팅 후킹 포인트:** 이 게이지가 스토어 GIF의 핵심. r이 줄어들며 중심이 밝아지는 20초 루프.

### 2-B. 자원 Readout — 좌측 패널

각 자원 행의 구성:
```
[아이콘] [자원 기호·이름]
         [현재값: 4.71e8]
         [생산율: +2.34e6/s]
```

- E(노란 ⚡), C(파란 ◎), D(보라 ▣), QF(녹 ◆) — 색상으로 즉시 구분
- QF는 생산율 대신 "영구 보존" 표시
- 생산율 = 0 이면 표시 안 함 (게임 초반 D, QF)

**천문학적 수치 처리:**
- 1e3 미만: 정수
- 1e3~1e6: K 접두사
- 1e6 이상: 과학 표기 (기본) 또는 접두사 전환 가능
- `break_eternity.js` Decimal → `.toStringWithDecimalPlaces(2)` 로 표기

### 2-C. 8단 체인 — 중앙 하단

컴팩트 테이블 형태. 행마다: **티어 배지 / 생산율 / 보유 수 / 비용 / 구매 버튼**

- T1~현재 해금 티어: 정상 표시
- 미해금 티어: 흐림(opacity 0.25~0.4), 잠금 버튼
- "구매" 버튼: E 충분하면 활성, 부족하면 흐림 (hover 시 "E 부족" 툴팁)
- 비용 증가 추세를 한눈에 읽을 수 있게 — 가장 유리한 구매 티어가 시각적으로 두드러지게

## 3. FTUE / 온보딩 — 첫 5분 흐름

**원칙: systems §4-1과 정확히 일치. 텍스트 벽 없이 시각으로 가르친다.**

### 3-1. 단계별 플로우

```
0:00  [화면 시작]
      "물질이 있다. / 압축하라."
      → 중앙에 반경 게이지 + 빨간 점 강조 [압축] 버튼만 보임
      → 자원 패널: E, C만 (D·QF 아직 숨김)
      → 체인 목록 없음

0:00~1:00  [클릭 단계]
      클릭마다 r이 즉시 줄어드는 피드백 (애니메이션, 수치 변화)
      E 소량 지급, C 증가
      → "Tier-1 압축기 구매 가능" 표시 (비용 강조)

1:00  [자동화 첫 해금]
      T1 구매 → 자동 C 생산 시작
      → 수치가 스스로 올라가는 것을 보여줌
      → [압축] 버튼 유지 (클릭 시 추가 이득 있음을 암시)

2:00  [원자층 진입 + 메커니즘 첫 등장]
      decade 1 도달 → 우측 패널에 "오비탈 공명" 위젯 등장
      슬롯 2개가 깜빡임
      텍스트: "오비탈이 열렸다." (단 한 줄, 클릭 유도 UI로 대체)

3:00~5:00  [공명 학습]
      슬롯 클릭 → 시각 피드백(슬롯 밝아짐) + "×1.5 배율" 수치 팝업
      D 첫 획득 → "연구" 탭 강조 점등 ("발견 데이터 축적. 연구 가능.")
      연구 탭 유도 → 탭 이동 선택지 제시 (강제 아님)
```

### 3-2. Progressive Disclosure 규칙

| 시점 | 공개 요소 | 숨김 유지 |
|---|---|---|
| 0~1분 | r 게이지, E, C, [압축] 버튼 | D, QF, 체인, 연구, 도감 |
| 1~2분 | T1 구매, T2 등장 | T3+, 도감, 상전이 |
| 2~3분 | 오비탈 공명 슬롯, D 첫 적립 | 연구 탭 (D 0이면 잠금) |
| 3~5분 | 연구 탭(D 획득 후), 도감 첫 항목 | B/C 가지, 상전이 탭 |
| dec19(프리온 벽) 최초 도달 | 상전이 탭 ● 첫 점등 (특별 강조 연출 — §5-6 참조) | 없음 (모두 표시됨) |
| dec19 이후 미지 벽 추가 도달 | 상전이 탭 ● 재점등 (일반 점등) | 없음 |

### 3-3. 온보딩 금지 사항

- 팝업 튜토리얼 오버레이 금지 (첫 클릭을 막지 않는다)
- 강제 순서 없음 (연구를 먼저 열어도 됨)
- 설명은 최대 1문장. 나머지는 UI 자체가 가르친다

## 4. 층별 메커니즘 UI 패턴

"한 층 = 한 새로움" — 새 메커니즘은 우측 패널에 점진적으로 추가된다. 기존 위젯은 사라지지 않고 축소 상태로 유지.

### 4-A. 분자층 (L1) — 기본 클릭

```
[우측 패널]
  현재 층: 분자층
  ─────────────────
  (메커니즘 위젯 없음 — 이 층의 새로움은 "r이 작아진다" 자체)
  도감 진행도: X/8
```

### 4-B. 원자층 (L2) — 오비탈 공명 슬롯

```
[우측 패널 · 오비탈 공명]
  ○ ○ ○  ← 슬롯 3개 원형 버튼
  활성 슬롯: 깜빡임(CSS animation) + 녹색 테두리
  슬롯 유효 시간: 3초 카운트다운 (작은 진행 링)
  클릭 시: 즉각 피드백 + "×1.5 배율 3초" 토스트
  자동화 후: 슬롯에 "AUTO" 표시, 수동 클릭 시 추가 보너스 남음
```

### 4-C. 핵층 (L3) — 핵력 게이지

```
[우측 패널 · 핵력 게이지]
  [████████░░] 82%  ← 수평 게이지 바
  충전율: +0.4%/s
  만충 시: 이벤트 발동 애니메이션 (바 깜빡임 → 방출 후 0% 초기화)
  과충전 노드 구매 후: 80% 마크에 점선 표시 (부분 방출 가능 지점)
```

### 4-D. 핵자층 (L4) — 색전하 3색 막대

```
[우측 패널 · 색전하 삼원합일]
  적 [████████░░] 82%
  청 [██████████] 100%
  녹 [█████░░░░░] 50%

  균형 비율: 0.50 → [패널티 아이콘]
  퍼펙트(≥0.9): [합일 달성] 표시 + 배율 수치 강조
  불균형: 막대 중 가장 낮은 것 빨간 테두리
```

### 4-E. 쿼크층 (L5) — 점근 자유 가속

```
[우측 패널 · 점근 자유]
  상태: 대기 중
  다음 발동까지: C 1.2e18 더 필요 (진행 바)
  발동 중: [AF 활성 — α 감소 중 · 14s 남음] (녹색 펄스 배경)
```

### 4-F. 프리온층 (L6) — 위상 상태 토글

```
[우측 패널 · 위상 겹침]
  [응집] [분산] [공명]  ← 3개 버튼 토글
  현재: 응집 (활성 강조)
```

### 5-6. 첫 상전이 FTUE — dec19 쿼크→프리온 (특별 강조)

첫 상전이(dec19)는 알려진 물리에서 미지 영역으로 건너가는 서사적 정점이다. 이 순간은 단순한 층 전환이 아니라 "지도가 끝난 곳"으로의 진입이므로, 탭 점등부터 화면 연출까지 일반 상전이보다 한 단계 강조한다.

**탭 점등 연출**

일반 상전이 점등이 정적인 ● 표시라면, 첫 상전이는 탭 레이블 전체에 펄스 애니메이션을 추가한다. 탭 배경이 1.5초 주기로 미세하게 밝아졌다 돌아오는 글로우를 반복하며, 소리가 켜진 환경에서는 저주파 틱 사운드를 1회 재생한다. 탭 레이블 텍스트는 "상전이 ●"가 아닌 "**미지 진입 ●**"로 일시 변경된다(상전이 화면 진입 또는 탭 클릭 시 원래대로 복귀).

**상전이 화면 차이점**

```
┌──────────────────────────────────────────────┐
│  [헤더] 미지 영역 진입 — 쿼크층 → 프리온층   │  ← 배경색 기본 배경 대신 딥 퍼플 그러데이션
│  +{N} QF 획득                  체류 {H}h {M}m │
├──────────────────────────────────────────────┤
│  [리셋/보존 매트릭스]  (일반 상전이와 동일)   │
├──────────────────────────────────────────────┤
│  [QF 첫 미리보기 — 강조 표시]                 │
│  "양자 거품 첫 획득 +{N} QF"                  │
│  QF 부스트 미리보기: 생산 × {mult}            │
│  (다음 런 시작 시 적용됨)                      │
├──────────────────────────────────────────────┤
│  [내러티브 비트 — 2줄 허용, 이 한 번만]        │
│  "알려진 물리의 경계를 넘는다.                 │
│   여기서부터 지도는 없다."                     │
├──────────────────────────────────────────────┤
│  [ 압축 계속 ]         [ 미지 영역으로 ]       │  ← 버튼 레이블 변경
└──────────────────────────────────────────────┘
```

일반 상전이 화면과 구조는 같되 세 가지가 다르다: (1) 헤더 배경 딥 퍼플 그러데이션으로 시각적 단절감 부여, (2) QF 미리보기 블록에 "첫 QF 획득" 강조 레이블 추가 — 이후 상전이에선 "추가 QF 획득"으로 돌아옴, (3) 내러티브 비트를 이 한 번만 2줄 허용(narrative.md §5-A의 미지 진입 메시지 사용). 확인 버튼 레이블도 "다음 층으로"가 아닌 "미지 영역으로"로 변경.

**FTUE에서 이 순간을 가르치는 방식**

온보딩은 이 순간을 직접 예고하지 않는다. 대신 쿼크층(dec9~19) 구간 동안 두 가지 간접 신호를 심어 플레이어가 스스로 낌새를 느끼게 한다. 첫째, 우측 패널 층 카드에 dec15 도달 시 "알려진 입자 경계 근접" 한 줄 상태 텍스트가 조용히 등장한다. 둘째, 도감 층 탭에서 "알려진 물리" 섹션의 마지막 입자들이 채워지면서 섹션 하단에 회색으로 잠긴 "미지 영역 —?" 슬롯이 흐릿하게 보인다. 이 두 단서가 dec19에서 탭 글로우 점등과 "미지 진입 ●" 레이블을 만났을 때, 플레이어는 설명 없이도 "뭔가 다른 일이 벌어진다"고 인지한다. 클릭 후 헤더 그러데이션과 2줄 내러티브가 그 인지를 확인해 주는 구조다.

### 5-7. CONCERN-3 능동 유도 — dec2~3 험프 넛지

qa-report CONCERN-3: dec2~3 구간은 자동 생산이 클릭을 완전히 대체하기 전, 공명 슬롯 타이밍을 놓치면 진행이 체감상 느려지는 구간이다. qa는 이를 제거할 버그가 아니라 "능동 텐션 구간"으로 규정했다. UX는 이 구간에서 플레이어를 몰아붙이는 대신, 개입 시 이득을 시각적으로 더 선명하게 보여주는 넛지를 배치한다.

**넛지 1 — 공명 슬롯 강조 증가**

dec2~3 진입 시 공명 슬롯의 깜빡임 속도를 평소보다 20% 빠르게 한다. 슬롯 유효 시간 카운트다운 링이 표준(3초)에서 2.5초로 단축되며, 이 값은 플레이어가 "더 급하게" 느끼도록 의도된 것이다. 자동화가 없는 이 구간에서 슬롯 타이밍이 핵심 개입 행위임을 시각적으로 상기시킨다.

**넛지 2 — 진행 속도 비교 힌트**

좌측 패널 상태 로그 영역에 dec2~3 구간 한정으로 1줄 힌트가 등장한다:

```
상태 로그 (4줄)
...
[힌트] 공명 클릭 시 현재보다 ×1.4 빠름
```

이 힌트는 실시간 계산(현재 C 속도 대비 공명 활성 시 속도 비율)을 기반으로 하며, 자동화 연구 노드 구매 후 또는 dec4 진입 시 사라진다. 플레이어를 몰아붙이는 경고가 아니라 정보 제공이므로 색상은 강조(녹/노)가 아닌 중립 회색 텍스트로 표시한다.

**넛지 3 — 체인 구매 유도 화살표**

dec2 구간에서 8단 체인 테이블의 "가장 유리한 티어" 행에 소형 화살표 아이콘(▶)을 표시한다. 이는 기존 §2-C에서 언급한 "가장 유리한 구매 티어 시각적 강조"의 구체적 구현이다. dec2~3 험프 구간에서 체인 투자 타이밍을 놓치지 않도록 유도하는 역할을 한다.

## 7. "작아짐=강해짐" 역설의 화면 구현

### 7-1. r 게이지 역설 메커니즘

| 요소 | r이 작아질수록 변화 |
|---|---|
| r 수치 (텍스트) | 지수 감소 (3.8→2.1→0.4×10⁻¹¹) |
| 중심 글로우 반경 | 증가 (3px → 8px → 14px) |
| 게이지 호弧 길이 | 층 안에서 채워지다 상전이 시 리셋 |
| 동심원 개수 | 층이 깊어질수록 외부 원 소멸 (원이 사라지며 "더 깊어짐" 시각화) |
| 배율 수치 | 커짐 (×1.0 → ×1.41 → ×2.94) |

**핵심:** r 수치와 중심 글로우는 **동시에 화면에 있다**. 플레이어는 "숫자는 0에 가까운데 빛은 커진다"는 역설을 매 틱 목격.

### 7-2. 자원 패널의 역설

- C(압축 깊이)는 커지지만 화면엔 보이지 않는다 — r만 보인다 (GDD §5 주의)
- 생산 배율(×1.41, ×1.60...)은 좌측 패널에서 커짐
- QF 양자 거품은 축적되어 강함을 수치로 표현

### 7-3. 층 전환 시 "강해짐" 연출

상전이 직후:
1. r 게이지가 잠깐 크게 확대되었다가 새 층 스케일로 재설정
2. 동심원 재배열 (새 층에 맞는 구조)
3. 우측 패널에 새 메커니즘 위젯 슬라이드인
4. 층 진입 내러티브 비트 1회 표시 (narrative.md §4)
