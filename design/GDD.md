## 1. 한 줄 컨셉
> 물질을 압축할수록 크기는 0을 향해 작아지지만, 그 대가로 미립자 세계의 힘이 쌓인다.
> 분자에서 시작해 플랑크 길이까지 내려가며 미지의 입자를 발견하는 방치형 인크리멘탈.

기존 인크리멘탈의 두 관습("숫자가 커진다" + "우주 스케일로 확장")을 모두 뒤집음: **숫자가 작아지고(테마) + 미립자 세계로 하강.** 단, 내부 통화는 커지되 화면의 반경 r이 작아지는 구조라 도파민은 유지된다.

## 2. 장르 / 정체성 / 플랫폼
- 장르: 아이들러 / 인크리멘탈 / 방치형
- 페이싱: **순수 방치 / 장기(하드코어).** 오프라인 진행이 1급 시민. (계보: Antimatter Dimensions, Universal Paperclips)
- 정체성: **B. 입자 실험실** — 플레이어는 연구소장(Director). 압축으로 미지의 입자를 발견. 메인 후킹 = 입자 도감 + 연구 트리. (세계관 [§14](#14-세계관--톤))
- 플랫폼: 웹(HTML/JS/TS) 프로토타입 → 최종 **PC Steam**. 패키징 = **NW.js + steamworks.js** (Tauri는 Steam 오버레이 미지원이라 제외 — 리서치 확정, §11). 중간 단계는 itch.io/galaxy.click 배포 가능.

## 3. 디자인 필러 (절대 안 어김)
1. **작아짐 = 강해짐.** 모든 시스템이 이 역설을 강화.
2. **테마 = 메커니즘.** 숫자·자원은 실제 물리에 뿌리. 장식 금지.
3. **방치도 보상, 개입도 보상.** 안 만져도 진행, 만지면 가속.
4. **한 층 = 한 새로움.** 스케일 한 칸 내려갈 때마다 새 메커니즘 1개.

## 4. 코어 루프
- **초 단위(능동, 선택):** 압축 → 반경 `r`↓, 에너지 `E`↑
- **분 단위(전술):** `E`로 압축기·연구 구매 → 효율↑, 새 입자 해금
- **시간 단위(전략):** 스케일 장벽(미지 서브층 벽) 도달 → 상전이 → 새 층/입자군/영구 부스트(QF) — *첫 상전이 ~4h 능동(목표 4~6h) / 엔드게임 호흡 ~2.5주 캘린더(빅 크런치 재하강 3회 = 2.47주)*
- **방치:** 자동 압축기 체인이 계속 생산 (오프라인 포함)
- **핵심 원칙 — "층 진입 ≠ 상전이":** 알려진 물리 5층(분자~쿼크)은 **상전이 없는 온보딩**(층 진입 = 스케일+도감+내러티브). **상전이는 미지 서브층 6개 + 빅 크런치(총 7회)에서만** 발생. ([§9](#9-레벨링-층--상전이))

## 5. 자원
| 자원 | 기호 | 생산원 | 용도 |
|---|---|---|---|
| 압축 에너지 | `E` | 압축기 체인 | 압축기·자동화 구매 |
| 압축 깊이 | `C` | E 소비(또는 체인 직접) | 반경 `r` 감소 |
| 발견 데이터 | `D` | 새 입자 발견 | 연구 트리 |
| 양자 거품 | `QF` | 상전이(prestige) | 영구 배율 |

> 주의: C는 천문학적으로 커지되 화면엔 r만 보인다 = "작아짐=강해짐" 필러의 구현. `lifetime_C`(전 런 누적 C)가 QF 산출의 기준(AdCap식, §7).

## 6. 엔진: 8단 압축기 체인 (Antimatter-Dimensions 골격)
- 압축기 Tier 1~8. **Tier k가 Tier k−1을 생산**, Tier 1이 압축 깊이 `C`를 생산 → 자기증식 가속.
- 8단 `t^n/n!` 성장이 지수 비용에 따라잡히는 시점 = 자연스러운 상전이 벽(별도 하드게이트 불필요, 시뮬 실증 §8).
- BigNumber 필요 (break_eternity.js) — C/개수가 float 한계를 넘음(재하강 누적 lifetime_C 포함).

## 7. 핵심 수식 (시뮬 검증 — `economy.md` v0.2)
```
반경:    r = r₀ · 10^(−dec),     r₀ = 1e-9 m (분자)
매핑:    dec = α · log₁₀(C + 1),  α = 0.65   ← 로그 저항: 작아질수록 기하급수로 어려움
생산:    dgₖ/dt = g_{k+1} · mult        (k=1..7)
        dC/dt  = g₁ · mult
비용:    cost_k(n) = base_k · growth_k^n
        base_k   = 10^(1 + 1.3·k)               ← v0.1 10^(1+2k)에서 하향: 2k는 온보딩 13h로 과도(이산 시뮬) → 1.3k로 1.2h 정상화 (economy §2.1·§2.2)
        growth_k = 2.2 − (0.4/7)·(k−1)          ← 내림차순 T1=2.2 … T8=1.8 (하위 가파름·상위 완만)
상전이:  QF_total       = floor( K · (lifetime_C / D_norm)^0.5 ),  K=1, D_norm=1e26 (제곱근)
        production_mult = 1 + 0.25 · log₁₀(1 + QF_total)         ← QF가 직접 힘이 됨(필러①)
오프라인: offline = rate × min(elapsed, CAP) × modifier × long_idle_bonus
        CAP=24h, modifier=0.65 (상전이 직후 첫 런=1.0), 48h 탬퍼 클램프, LB=0.5 로그보너스
```
- `dec = α·log₁₀(C)` 핵심 발견: `log₁₀(C)`가 시간에 대해 concave → 후반 호흡이 기하급수로 길어짐. 소프트캡·인위적 제동 불필요. 물리적 정합("작아질수록 저항 기하급수").
- 비용 곡선: growth 1.07~1.15(업계 표준)는 우리 체인엔 폭주 유발 → **2.2~1.8 대역**으로 magnitude 상향, 내림차순 형태만 유지. 마일스톤 없이도 최적 구매 티어가 시점마다 자연 이동(검증).
- 상전이: 제곱근(e=0.5, AdCap형) 채택. QF→부스트 직결(`1+0.25·log₁₀`)로 "작아짐=강해짐" 구현 + 후반 자기감쇠. base_rate=0.25는 race 안전 구간(0.5는 붕괴). **QF는 `lifetime_C` 단일 모델**(층별 계수 미사용).
- 오프라인: 등가-시간 일괄지급(보수적 하한). 48h 하드 클램프로 시계조작 차단.
- 미검증: 구매 휴리스틱(고티어 우선 가정) → 프로토타입에서 실측 시 페이싱 ±20% 가능.

## 9. 레벨링: 층 / 상전이

**층 = 11개(알려진 5 + 미지 6).** 상전이 = **7회**(미지 6 + 빅 크런치). decade는 **26(플랑크)에서 캡**. 상세 `systems.md` v1.1.

| 층 | 서사 스케일(측정) | 게임플레이 벽 | 새 메커니즘 (한 층=한 새로움) | 상전이 |
|---|---|---|---|---|
| 분자 | 1e-9 m | 온보딩 | 수동 압축 클릭 | — |
| 원자 | 1e-10 | 온보딩 | 전자 오비탈 공명 | — |
| 핵 | 1e-14 | 온보딩 | 핵력 게이지 | — |
| 핵자 | 1e-15 | 온보딩 | 색전하 삼원합일 | — |
| 쿼크 | 1e-18 | 온보딩(꼬리 dec9~19) | 점근 자유 가속 | — |
| [미지] 프리온 | 1e-19~1e-21 | dec19 (4.3h) | 위상 겹침 | **PT1 (첫 상전이)** |
| [미지] 끈 | 1e-21~1e-24 | dec21.5 | 진동 하모닉스 | PT2 |
| [미지] 루프 | 1e-24~1e-27 | dec23 | 스핀 네트워크 | PT3 |
| [미지] 거품 | 1e-27~1e-30 | dec24.5 | 불확정 요동 | PT4 |
| [미지] 정보 | 1e-30~1e-33 | dec25.5 | 홀로그래픽 인코딩 | PT5 |
| [미지] 플랑크 | 1e-33~1.6e-35 | dec26 (캡) | 시공간 픽셀 붕괴 | PT6 + **빅 크런치(PT7)** |

- **서사 스케일(측정) ≠ 게임플레이 벽(상전이).** 측정 스케일은 도감/내러티브 표기, 압축 장벽은 dec19~26 대역. 두 좌표계 분리(디렉터 중재, `director-review.md`).
- **알려진 물리 5층(dec0~9 + 쿼크 꼬리 dec9~19) = 무상전이 온보딩**(첫 1.2h). 메커니즘은 온보딩에서 학습. 층 진입 = 스케일+도감+내러티브 비트.
- **미지 서브층 6개 = 상전이 1:1.** WALLS=[19, 21.5, 23, 24.5, 25.5, 26], 각 서브층 체류 **≥4h** 보장(시뮬·QA 검증) → "한 층=한 새로움" 구조적 강제.
- 첫 상전이 = **쿼크→프리온(dec19)** = 미지 영역 진입의 서사적 정점.
- **빅 크런치(PT7) = 플랑크 도달 → 최종 QF 폭발 → 재하강 루프 시작점.** 엔드게임이 아니라 *반복 메타 루프의 입구*. 재하강은 QF 부스트로 미지 서브층을 빠르게 재통과(systems §2-J), "이번엔 어느 서브층을 깊이 파는가"의 메타.
- 리셋/보존: E·C·압축기 리셋 / QF·도감·연구 영구 보존 / D 부분 보존(보존율 후속). 상세 매트릭스 `systems.md` §5-2.

## 12. 시장 차별화 (리서치 확정 — `research/theme-competitors.md`)
경쟁작 **16종** 조사: 숫자가 실제로 작아지는 게임은 **Universe Shrinker 하나뿐**(그마저 우주 스케일). **"분자→플랑크 미립자 하강 + 압축=자원 + 입자 도감" 조합은 시장 완전 공백.**
- Steam 경쟁: Criticality(분열, 88% 긍정), Incremental Mass Rewritten(질량 증가, Q4 2026 예정) — 둘 다 *커지는* 방향.
- 배울 점: 압축 청각 피드백(Criticality), 층마다 메커닉 재구성(Fundamental), 압력/온도 소프트캡(Elemental).
- 피할 함정: "여러 층 동안 새 것 없음"(Universe Shrinker), 불투명 공식(Incremental Mass), 상향 전환 유혹 → **하강 방향 + 한 층=한 새로움 일관 유지**.

## 14. 세계관 / 톤 (`narrative.md` v0.2)
- **설정:** "Compression Institute" — 지하 깊은 입자 압축 연구소. 플레이어 = 익명의 연구소장. 공식 목적은 미지 입자 탐색, 실제 동기는 쿼크 이하에서 잡힌 옛 신호의 정체.
- **보이스 한 줄:** *"압축의 로그 — 냉정하고 정확하되, 발견 앞에서는 경이를 숨기지 않는다."* 물리 용어 정확히, 감탄사 없음(느낌표는 빅 크런치 1회), AI 슬롭 금지.
- **발견 아크:** 확인(알려진 물리) → 경이 → 충격 → 침묵 → 붕괴(플랑크). 미지로 갈수록 연구소장의 목소리가 신중·간결해짐.
- **트리거 분리:** 층 전환 비트(11개, 스케일 진입) ≠ 상전이 메시지(7개, 벽 도달). 쿼크→프리온만 둘이 결합(첫 상전이).
