## 1. 비주얼 아이덴티티

### 1-A. 아트 스타일 정의

**"실험 기록의 미학(Instrument Aesthetic)"**

이 게임의 룩은 연구 기기와 측정 도구에서 온다. 오실로스코프 디스플레이, 입자 검출기 출력 그래프, 전자현미경 이미지의 화면 언어. 화려하지 않다. 정밀하다. 수치와 좌표가 아름다운 것은 그것이 진짜이기 때문이다.

**거부하는 룩**
- 네온 사이버펑크: 파란 그리드 + 보라 글로우 + 핑크 아우트라인. 이 미감은 "미래"의 상투어다. 이 게임이 다루는 물리는 상상이 아니라 현실이다.
- 우주적 장엄함(space epic): 성운 텍스처, 별 배경, 황금빛 행성. 이 게임은 위를 보지 않는다. 아래로 내려간다.
- 귀여운 인크리멘탈: 파스텔, 둥근 아이콘, 클리커 게임 미감. 냉정한 연구소장의 보이스와 불일치.

**채택하는 룩**
- 계측기 그린(Phosphor Green): 오래된 CRT 모니터, 레이더 스크린. 초록빛 발광이 어둠 안에서 데이터를 읽어낸다.
- 실험실 다크 베이스: 거의 검은 배경(#0a0c0f ~ #0e1215). 빛이 정보다. 배경은 침묵이다.
- 과학 문서 타이포그래피: 모노스페이스가 수치를 지배하고, 산세리프가 레이블을 담당한다. 이탤릭과 기울임은 없다.
- 물리적 광량: 각 층의 빛은 실제 물리 현상에 근거한다. 분자층의 분산광, 쿼크층의 강한 결합 에너지, 플랑크층의 진공 붕괴.

## 2. 층별 비주얼 언어

### 2-0. 설계 원칙

각 층은 **색·질감·광량·모션**의 4축으로 구분된다.

- **색**: 층의 악센트 컬러. 배경 딥 위에 올라가는 유일한 유채색.
- **질감**: 배경과 게이지의 노이즈·패턴 특성.
- **광량**: 환경광의 밝기와 산란 반경. 깊을수록 광원은 작아지고 강해진다.
- **모션**: 앰비언트 파티클의 속도·밀도·형태.

"작아질수록 거대한 진실" — 층이 깊어질수록 화면은 더 어두워지지만 중심의 빛은 더 강하고 날카로워진다.

### 2-2. 층별 전환 — 색온도 아크

분자 → 핵까지: **따뜻한 색 계열** (황록·청록·주황). 알려진 물리의 온기.

핵자 → 쿼크: **삼원색 → 탈색**. 색전하의 화려함이 점근 자유에서 색을 잃는다.

프리온 이하: **차갑고 날카로운 단색 계열** (자주→적→남→은→청백→백). 미지는 차갑다. 그리고 점점 더 순수해진다.

플랑크: 색이 없다. 있다면 모든 색이 합쳐진 **순백**이다. 시공간이 픽셀이었다.

### 2-3. 배경 질감 상세

질감은 graphics-programmer와 협의하여 Canvas2D 또는 WebGL로 구현한다. 아래는 방향 스펙.

**분자층 (L1)**: 저주파 Perlin noise 오버레이. 투명도 0.04~0.06. 초록 계열 앰비언트 헤이즈. 움직임은 0.3px/s 이하의 드리프트.

**원자층 (L2)**: r 게이지 중심에서 방사하는 동심원 패턴. 8~12개 원, 가장자리로 갈수록 투명도 감소. 펄스 주기 4~6초.

**핵층 (L3)**: 중심 영역 클러스터 노이즈. 반경 30% 이내 밀도 집중. 외곽은 빠른 페이드.

**핵자층 (L4)**: 삼각형 분할 기반 패턴. RGB 각각 다른 삼각형 클러스터. 색전하 균형에 따라 색비율 실시간 변화.

**쿼크층 (L5)**: 텍스처 최소화. 배경 거의 순수 딥 다크. 파티클만 존재. 점근 자유 활성 시 흰 선형 파티클이 짧은 궤적을 그리고 사라짐.

**프리온층 (L6)**: 위상 간섭 패턴. 두 파원 간 보강/상쇄 무늬. 느리게 이동하는 마디선. 위상 토글에 따라 패턴 즉시 변화.

**끈층 (L7)**: 수평 사인파 필라멘트 다수. 진폭·주기가 하모닉 번호에 따라 다름. 색은 진동 에너지에 비례하는 열지도(heat map).

**루프층 (L8)**: 6방향 격자 선. 노드 점에서 주기적 펄스 방사. 스핀 네트워크 캔버스 위젯(ux.md §4-H)과 비주얼 언어 일치.

**거품층 (L9)**: 화이트 노이즈 기반 미세 버블 생성·소멸. 버블 반경 2~8px, 수명 0.3~2.0초, 무작위 위치. 요동 이벤트 발생 시 해당 영역 버블 밀도 순간 증가.

**정보층 (L10)**: 대각선 스캔라인 패턴 + 홀로그램 격자. 화면 전체에 얇은 수평선(투명도 0.06). D 누적량에 비례해 스캔라인 밀도 증가.

**플랑크층 (L11)**: 배경 전체 픽셀 그리드(1~2px 단위). 게이지 진행에 따라 그리드 선이 서서히 뚜렷해짐. dec26 도달 직전 픽셀 단위 화면 해체 이펙트.

## 3. 스케일 하강 연출

### 3-A. 압축 줌 (매 틱)

r 게이지 중심 글로우가 커지는 것이 메인 줌 피드백이다. C 값 증가에 따라:

- 중심 글로우 반경: `3px → 8px → 14px` (ux.md §7-1 기준). 선형 전환(200ms ease-out).
- 동심원: 외곽에서부터 순차 페이드아웃. 층이 깊어질수록 남은 원이 줄어든다(층 내에서는 유지, 상전이 시 소멸).
- 글로우 색: 베이스 `--col-glow-core` (#3ecf8e)에 층별 악센트 컬러를 10~20% 블렌딩.

**그래픽스 프로그래머 협의 필요**: 글로우를 Canvas2D `radialGradient`로 구현할지 WebGL fragment shader로 구현할지. 후자는 더 자연스러운 광량 제어가 가능하나 성능 예산 확인 필요.

### 3-B. 층 경계 돌파 — 알려진 물리 5층

상전이 없는 층 전환(분자→원자→핵→핵자→쿼크)의 비주얼:

1. **페이드 아우트**: 현재 층 배경 질감이 0.3s에 걸쳐 투명도 0으로.
2. **스케일 파열**: r 게이지가 0.2s 안에 120%로 팽창 후 새 층 스케일로 즉시 수축. 물리적으로 "층 안으로 뚫고 들어가는" 느낌.
3. **악센트 컬러 전환**: 0.5s 선형 전환. 갑작스럽지 않되 명확하게.
4. **층 카드 슬라이드인**: ux.md §7-3에 따라 우측 패널 새 층 카드가 아래에서 슬라이드인(200ms).

**파티클 리셋**: 이전 층 파티클 0.3s 페이드아웃, 새 층 파티클 0.5s 페이드인.

### 3-C. 상전이 경계 돌파 — 미지 서브층 (QF 획득)

상전이는 층 전환과 다른 무게를 가진다. 연구소장이 새로운 물리에 진입하는 순간이다.

**상전이 진입(탭 클릭 → 리셋 순간)**:

1. **화면 전체 화이트 플래시**: 0.05s, 투명도 0.15. 압축 에너지 방출의 시각적 은유.
2. **r 게이지 폭발 후 리셋**: 글로우가 전체 게이지 영역을 채울 때까지 팽창(0.4s) → 순간 소멸 → 새 층 스케일로 재등장(0.3s 페이드인).
3. **QF 카운터 증가 애니메이션**: QF 수치가 이전 값에서 새 값으로 0.6s에 걸쳐 증가. 숫자 롤링이 아닌 fade-cross(이전 값 페이드아웃, 새 값 페이드인). 동시에 QF 아이콘(◆) 1회 펄스.
4. **층별 악센트 컬러 전환**: 즉각 전환(디더링 없음). 단절감이 의도적이다.
5. **새 메커니즘 위젯 슬라이드인**: ux.md §4 메커니즘 위젯이 우측에서 슬라이드인(300ms).

**첫 상전이(쿼크→프리온, dec19) 추가 연출**:

- 전환 전 0.5s: 화면 전체 색이 서서히 탈색(desaturate). "알려진 물리"의 색이 빠진다.
- 화이트 플래시 강도 2배(투명도 0.3).
- 플래시 후 새 층(프리온) 악센트 컬러(#7c4dff)가 0.8s에 걸쳐 스며드는 fade-in.
- 헤더 배경: ux.md §5-6에 따라 딥 퍼플 그러데이션. 단, 그러데이션이 아닌 레이어드 솔리드(딥 다크 위 반투명 퍼플 레이어 #7c4dff at 0.15 opacity).

## 5. 게임필/주스 (시각 피드백 가이드)

### 5-A. 핵심 원칙

이 게임의 주스는 **절제된 정밀함**이다. 과도한 이펙트는 연구소장의 냉정한 보이스를 배신한다. 모든 피드백은 정보를 담거나, 물리 현상을 시각화하거나, 없어야 한다.

**금지**:
- 별 폭발, 코인 분수, 무지개 파티클 — 이것은 클리커 게임이 아니다.
- 화면 셰이크(게임 오버 불필요한 과장) — 진동은 층 메커니즘의 의미가 있을 때만.
- 1초 이상 지속되는 이펙트(단, 빅 크런치 제외) — 방치형은 배경이 조용해야 한다.

**허용**:
- 미세 진동(스쿼시·스트레치): 구매 버튼이 눌릴 때 0.95 scale → 1.0 (80ms).
- 단색 플래시: 층별 악센트 컬러, 최대 투명도 0.15, 최대 0.2s.
- 수치 팝업(floating text): 발견 보너스 등장 시 "+X%" 텍스트가 위로 이동하며 페이드.
- 아이콘 펄스: QF 획득 시 ◆ 아이콘 1.0→1.2→1.0 scale (0.3s).

### 5-B. 행동별 피드백 스펙

**클릭(압축 버튼)**:
- 버튼: scale 0.97 → 1.0 (60ms ease-out).
- r 게이지 중심 글로우: +2px 순간 확장 → 200ms 내 원래 값 복귀.

- 파티클: 게이지 중심에서 4~6개 방사형 파티클, 0.3s, 소형(3px), 층 악센트 컬러.
- 오비탈 공명 클릭(L2): 슬롯 0.5s 밝아짐 + 슬롯 테두리 #3ecf8e로 전환 + "×1.5" float text.

**구매(압축기)**:
- 버튼: scale 0.95 → 1.0 (80ms).

- E 부족 시 버튼: shake 애니메이션(3px, 2cycle, 150ms). 색: 테두리 `--col-reset` 잠깐.

**입자 발견**:
- 도감 카드 reveal wipe (800ms, 위→아래).
- 토스트: "[입자명] — 도감에 기록됨" (화면 우상단, 2s 표시, 층 악센트 컬러 테두리).

**상전이(미지 서브층 6회)**:

- 상전이 탭 ●: 정적 도트 → 들어올 때(1.5s 주기 펄스, 첫 상전이만).

### 5-C. 앰비언트 모션 — 방치 상태의 살아있음

방치형은 "가만히 둬도 아름다운" 화면이다. 플레이어가 자리를 비웠을 때도 화면은 살아있어야 한다.

**r 게이지**: 자동 압축 중이면 글로우가 매우 느리게(4s 주기) 맥동한다. 완전 정적이 아니다.

**배경 파티클**: 층별 앰비언트 파티클이 항상 움직인다. 속도는 C 생산률에 비례. 방치 중에도 살아있음을 보여준다.

**자원 수치**: 생산률 > 0이면 수치가 서서히 증가하는 것이 보인다. 업데이트 주기: 1~2초. 급격한 점프 없이 부드럽게.

**핵력 게이지(L3)**: 자동으로 충전 중이면 게이지가 천천히 차오른다. 방치 중 만충 이벤트가 발생하면 잠깐 플래시 — 화면 앞에 있든 없든 기록된다.

- 분자층: 황록 배경, 넓은 산란광, 큰 r 수치(`1.00×10⁻⁹ m`), 작은 글로우.
- 쿼크층: 거의 검은 배경, 선형 파티클, 매우 작은 r(`4.2×10⁻¹⁸ m`), 강한 코어.
- 플랑크층: 순흑 배경, 픽셀 그리드 균열, 극소 r(`1.7×10⁻³⁵ m`), 백색 코어 포화.

캡션(스토어 페이지 이미지 아래): *"같은 화면, 다른 세계."*

### 6-D. 후킹 샷 3 — "도감 발견" (스크린샷)

도감 탭 화면. 알려진 물리 섹션은 채워져 있고, 미지 섹션은 대부분 마스킹.

- 마스킹된 카드들 사이에서 하나가 막 reveal wipe 중인 순간을 포착.
- "██████" 마스킹과 "개방끈 모드-3" 텍스트가 공존하는 대비.
- 도감 완성도 바: 23/87. 미지 영역 진입 초입.

### 6-E. 후킹 샷 4 — "빅 크런치 순간" (스크린샷)

QF_BURST 화면의 정지 샷.

- 순흑 배경.
- 중앙 텍스트: "붕괴가 아니다. / 재압축이다."
- 흰 링 이펙트가 퍼지는 순간.
- 화면 우상단: "+{N} QF 획득" 수치.

이 샷은 단어 하나 없이도 "이 게임의 프레스티지는 다르다"를 전달한다.

### 6-F. 트레일러 방향

**30초 트레일러 구조** (비주얼 방향, 카피 없음):

0~5s: 분자층. 느린 압축. 황록 빛.
5~10s: 핵층 → 쿼크층 빠른 몽타주. 배경이 점점 어두워짐. 글로우가 커짐.
10~15s: 첫 상전이. 화이트 플래시. 심자주로 전환. 도감 카드 reveal 연속.
15~22s: 미지 서브층 4개 빠른 전환. 각각의 색과 질감이 다름. 층이 다른 세계임을 보여줌.
22~28s: 빅 크런치 시퀀스. BLACKOUT 3초 — 화면 암전.
28~30s: "재압축이다." 텍스트 등장. 흰 링. 로고.

**사운드**: audio-designer와 협력. 각 층 전환 시 음역 하강. 빅 크런치 = 침묵 후 단일 저음.

## 7. graphics-programmer 협의 항목

| 항목 | 우선도 | 내용 |
|---|---|---|
| r 게이지 글로우 렌더링 방식 | 높음 | Canvas2D radialGradient vs WebGL fragment. 60fps 보장 조건 확인 |
| 층별 배경 질감 구현 | 높음 | Perlin noise, 동심원, 격자 등 각 층 텍스처. 성능 예산 협의 |
| 파티클 시스템 최대 수 | 높음 | 앰비언트 + 인터랙티브 동시 파티클 상한. 레이어 분리 구조 |
| 상전이 플래시 이펙트 | 중간 | 화이트 플래시 타이밍·투명도. CSS transition vs Canvas overlay |
| 빅 크런치 링 이펙트 | 중간 | 흰 링 방사 애니메이션. Canvas 또는 SVG 구현 |
| 도감 카드 reveal wipe | 중간 | CSS clip-path animation 가능 여부. 또는 Canvas mask |
| 전설 입자 카드 맥동 | 낮음 | CSS animation으로 충분한지 확인 |
| 색전하 삼원색 파티클(L4) | 낮음 | 3색 삼체 시뮬레이션 성능 |
