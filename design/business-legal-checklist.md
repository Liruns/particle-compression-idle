### W-8BEN 핵심 (⭐ 전문가 확인 필요)

- **한국-미국 조세조약**: 한국은 미국과 조세조약 체결국. 한미 조세조약 제13조(로열티 조항) → 미국 원천소득에 대한 원천징수율 **최대 10%**(조약 미적용 시 30%).
- **Steam 수익의 성격**: Valve가 배분하는 개발자 수익은 "로열티"로 분류됨.
- **W-8BEN 제출 = 조약 혜택 적용 신청**: 제출하면 10%로 인하 가능(미제출 시 30%). 단 개인/법인 상황, TIN 여부, 소득 성격에 따라 달라질 수 있음.
- **TIN**: 외국 TIN = 한국 주민등록번호 또는 사업자등록번호. ITIN(미국 개인납세자번호)은 별도 신청 가능하나 한국 TIN으로 대체 가능한 경우 불필요(확인 필요).
- **1042-S**: Valve는 원천징수 발생 시 1042-S 서식 발행. 한국에서 외국납부세액공제 신청 가능(세무사 확인 필요).
- **유효기간**: W-8BEN은 서명일 기준 약 3년 유효. 주기적 갱신 필요.

> **세무사/법무 전문가 확인 필요**: 개인사업자 등록 여부·소득 분류(사업소득/기타소득)·ITIN 필요 여부·외국납부세액공제 처리 방법은 세무사에게 확인.
> 공식 출처: [Steamworks 세금 FAQ](https://partner.steamgames.com/doc/finance/taxfaq?l=koreana) / [IRS 조세조약 목록](https://www.irs.gov/businesses/international-businesses/united-states-income-tax-treaties-a-to-z) / [한미조세조약(홈택스)](https://txsi.hometax.go.kr/docs_new/customer/law/statutePact_jomun_detail.jsp?inter_law_id=116&gubun=3)

### 간이과세 vs 일반과세 판단 포인트 (⭐ 세무사 확인 필요)

| 구분 | 간이과세 | 일반과세 |
|---|---|---|
| 2025년 기준 연매출 | 1억 400만원 미만 | 1억 400만원 이상 |
| 부가세율 | 1.5~4% (업종별) | 10% |
| 부가세 면제 | 연 4,800만원 미만 시 납부 면제 | 해당 없음 |
| 세금계산서 | 4,800만원 미만 발급 불가 / 이상 발급 의무 | 의무 발행 |
| 매입세액 공제 | 공제율 낮음 (0.5%) | 전액 공제 |
| 부가세 신고 주기 | 연 1회 | 연 2회 |

- 초기 해외 게임 출시(Steam 전용)는 영세율 적용 가능성 있으므로 매출 규모에 따라 일반과세 선택이 유리할 수 있음.
- Steam 해외 매출 = 영세율 가능성 있으나 서류 요건(비거주자·외국법인에 공급 증빙) 확인 필요.

> **세무사 확인 필요**: 업종코드 선택, 간이/일반 판단, 영세율 적용 여부.
> 공식 출처: [국세청 부가가치세 안내](https://www.nts.go.kr/nts/cm/cntnts/cntntsView.do?cntntsId=7693&mi=2272) / [간이과세 기준 2025(청구스 블로그)](https://www.chungoose.kr/blog/2025년-간이과세자-기준-정리-8-000만원-1억-400만원)

## 3. ⭐⭐ GRAC 게임물 등급분류 (가장 자주 놓치는 법적 게이트)

> **이 항목이 가장 위험하다.** 등급분류를 받지 않고 한국 내 서비스를 제공하면 게임산업진흥법 위반.

### 3-A. 현행 법적 구조 (2026년 6월 기준)

게임산업진흥에 관한 법률 제21조: 게임물을 유통·서비스하려는 자는 사전에 등급분류를 받아야 함.

| 등급분류 기관 | 대상 | 비고 |
|---|---|---|
| **게임물관리위원회(GRAC)** | 청소년이용불가(19금) 게임 (PC·콘솔 포함) | grac.or.kr |
| **게임콘텐츠등급분류위원회(GCRB)** | 전체이용가·12세·15세 이용가 (PC·콘솔, 2025년 11월부터 청불 등급도 취급) | gcrb.or.kr |
| **자체등급분류사업자** | 승인된 플랫폼(구글플레이, 앱스토어, 원스토어, 마이크로소프트 스토어 등)이 IARC 시스템으로 자동 처리 | Steam은 해당 안 됨(아래 참조) |

### 3-B. ⭐ Steam은 자체등급분류사업자가 아니다 (결정적 포인트)

**핵심 사실**: Steam(Valve)은 현재 한국의 자체등급분류사업자(IARC 연계 플랫폼)가 **아니다**.
- 구글 플레이, 애플 앱스토어, 마이크로소프트 스토어 등은 IARC 시스템을 통해 자동 자체등급분류 가능.
- Steam은 IARC 미가입 상태(2026년 6월 기준). GRAC 공식 입장: "밸브에 한국 규제 변경 안내 및 자체등급분류사업자 등록을 독려 중".
- 즉, **Steam을 통해 한국 이용자에게 서비스하는 경우, 개발자가 직접 GRAC 또는 GCRB에 등급분류 신청해야 할 가능성이 높다.**

### 3-C. 실무적 현황 (주의: 그레이 영역)

**현재 업계 현황**: 많은 해외 개발자가 Steam을 통해 한국에 등급분류 없이 게임을 판매 중. GRAC이 이를 모두 단속하지는 않고 있음. 단:
- 한국어 로컬라이제이션을 지원하거나 한국 이용자를 명시 타깃으로 하는 경우 = 리스크 증가
- 한국 개발자(내국인)가 자국 법을 위반한 경우 = 단순 외국 개발자보다 리스크 높음
- 게임 등급분류 없이 서비스 적발 시 징역 2년 이하 또는 벌금 2천만원 이하(게임산업진흥법 제44조)

**추천**: 우리 게임은 한국 개발자가 한국 이용자에게도 판매할 것이므로 **직접 등급분류 신청을 강력 권고.**

> **법무/전문가 확인 필요**: 현재 Steam 출시 게임의 GRAC 등급분류 의무 여부 + 최신 행정 해석. GRAC 콜센터(1566-2941) 또는 법무 전문가에게 직접 문의.

### 3-D. GRAC/GCRB 직접 신청 절차 (방치형 게임 기준)

우리 게임은 방치형 인크리멘탈로 성인 콘텐츠 없음 → "전체이용가" 또는 "12세이용가" 예상 → **GCRB 신청 대상**.

| # | 단계 | 내용 | 소요 시간 |
|---|---|---|---|
| G-01 | 사업자 등록 완료 | GRAC/GCRB는 업체 회원 등록 필요 (개인도 가능하나 사업자 권장) | B-01 완료 후 |
| G-02 | GCRB 회원 가입 | gcrb.or.kr에서 업체 회원 등록 | 1~2일 |
| G-03 | 등급분류 신청서 작성 | 신청인 정보, 게임 정보(플랫폼: 기타 PC/Steam), 게임물 내용정보 작성 (선정성/폭력성/약물/언어/사행성 각 항목 체크) | 당일 |
| G-04 | 첨부 서류 준비·업로드 | 사업자등록증, 게임물내용설명서, 실행 가능한 게임물 파일, 플레이 동영상 | 2~3일 |
| G-05 | 심의 수수료 납부 | 규모에 따라 상이 (GCRB 공식 수수료 기준 확인 필요) | 납부 후 심의 개시 |
| G-06 | 심의 결과 대기 | 약 10~15일 소요 (GRAC 기준, GCRB 유사) | 10~15일 |
| G-07 | 등급분류증명서 수령 | 스토어 페이지 및 게임 내 표기에 활용 | 심의 완료 후 |

**예상 등급**: 전체이용가 (폭력성·선정성 없음, 사행성 해당 없음 — 압축 에너지는 현금거래 불가).

> **주의**: 게임 내용이 변경되어 등급분류 기준을 초과하는 요소가 추가되면 재분류 신청 필요.
> 공식 출처: [게임콘텐츠등급분류위원회](https://www.gcrb.or.kr/) / [게임물관리위원회](https://www.grac.or.kr/) / [게임물 등급분류 안내(정부24)](https://www.gov.kr/portal/service/serviceInfo/PTR000051811) / [Toss 앱인토스 게임등급분류 가이드](https://toss.im/apps-in-toss/blog/game_rating_classification)

## 7. 전문가 확인이 필요한 최우선 항목 3개

1. **GRAC/GCRB 등급분류 의무**: Steam으로 한국 이용자에게 게임을 판매하는 한국 개발자에게 적용되는 현행 행정 해석. GRAC 콜센터(1566-2941) 또는 게임 전문 법무사 확인.

2. **W-8BEN + 원천징수 10%**: 한미 조세조약 로열티 조항 적용, TIN 기재 방법, 외국납부세액공제 절차. 세무사 확인.

3. **Steam 매출 부가가치세 영세율 적용**: 해외 이용자 매출 영세율 서류 요건, Valve의 VAT 대리 납부가 개발자 신고 의무를 면제하는지 여부. 세무사 확인.
